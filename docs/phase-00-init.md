# Fase 0 - Kickoff & Documentação Inicial

## Objetivo do Projeto
Criar uma Landing Page moderna e minimalista para o app "Lumen Reader", com foco em conversão (download) e apresentação estética premium (Glassmorphism).

## Requisitos Principais
1. **Design**: Glassmorphism, Neon Blue/Violet, Dark Mode.
2. **Funcionalidade**: Botão de Download dinâmico via GitHub API.
3. **Páginas**: Home, Privacidade, Termos.
4. **Navegação**: Router Client-Side para experiência fluida (Single Page Application feel).

## Stack Tecnológica
- **Core**: HTML5, Vanilla JavaScript.
- **Build Tool**: Vite (para otimização e hot-reload).
- **Estilização**: CSS Nativo (Variables, Glassmorphism).
- **Deploy**: Pronto para Vercel/Netlify.

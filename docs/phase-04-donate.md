# Fase 4 — Feature: Página de Doação (Donate)

## Descrição
Implementação de uma página dedicada para suporte ao projeto, permitindo que usuários contribuam financeiramente para a manutenção do Lumen Reader.

## Ações Realizadas
1. **Criação do Componente `Donate.js`**:
   - Design Premium com Glassmorphism.
   - Opção de doação focada em usuários brasileiros: **PIX**.
   - Funcionalidade de "Clique para copiar" para a chave PIX.


   
2. **Integração no Roteador**:
   - Rota `/donate` registrada no `main.js`.

3. **Atualização da Navegação**:
   - Link "Apoiar" adicionado à `Navbar.js` com destaque visual (cor de acento).
   - Link "Apoiar Projeto" adicionado ao `Footer.js`.

4. **SEO & Indexação**:
   - Adicionada a URL `/donate` ao `sitemap.xml`.

## Design System
- **Cores**: Gradientes lineares (Inter -> Accent Blue).
- **Interação**: Hover effects nos cards de doação e feedback visual na cópia da chave PIX.
